## Prerequisites

You need a C++20-capable compiler:

- GCC 12 or later
- Clang 15 or later
- MSVC 19.36 or later

No build system, package manager, or additional dependencies are required.

## Hello world

The following example is the canonical reflect overview. It demonstrates every core API function in a single file that compiles and runs with no changes.

<Steps>
  <Step title="Get the header">
    Download the single header file directly:

    ```bash
    curl -O https://raw.githubusercontent.com/qlibs/reflect/main/reflect
    ```

    Or copy it from `qlibs/reflect` into your project's include directory. The file has no extension — it is included as `<reflect>`.
  </Step>

  <Step title="Write your program">
    Create a file `hello.cpp` with the following contents:

    ```cpp
    #include <reflect>
    #include <print>

    using namespace std::literals;

    enum E { A, B };
    struct foo { int a; E b; };

    constexpr auto f = foo{.a = 42, .b = B};

    // reflect::size — number of members in an aggregate
    static_assert(2 == reflect::size(f));

    // reflect::type_id — unique compile-time identifier per type
    static_assert(reflect::type_id(f.a) != reflect::type_id(f.b));

    // reflect::type_name — string_view of the type's name
    static_assert("foo"sv == reflect::type_name(f));
    static_assert("int"sv == reflect::type_name(f.a));
    static_assert("E"sv   == reflect::type_name(f.b));

    // reflect::enum_name — string_view of an enumerator's name
    static_assert("B"sv == reflect::enum_name(f.b));

    // reflect::member_name — string_view of a member's name by index
    static_assert("a"sv == reflect::member_name<0>(f));
    static_assert("b"sv == reflect::member_name<1>(f));

    // reflect::get by index
    static_assert(42 == reflect::get<0>(f));
    static_assert(B  == reflect::get<1>(f));

    // reflect::get by name
    static_assert(42 == reflect::get<"a">(f));
    static_assert(B  == reflect::get<"b">(f));

    // reflect::to — convert aggregate to another type (e.g. std::tuple)
    constexpr auto t = reflect::to<std::tuple>(f);
    static_assert(42 == std::get<0>(t));
    static_assert(B  == std::get<1>(t));

    int main() {
      // reflect::for_each — iterate over members with a compile-time index
      reflect::for_each([](auto I) {
        std::println("{}.{}:{}={} ({}/{}/{})",
          reflect::type_name(f),                  // foo, foo
          reflect::member_name<I>(f),             // a  , b
          reflect::type_name(reflect::get<I>(f)), // int, E
          int(reflect::get<I>(f)),                // 42 , B
          reflect::size_of<I>(f),                 // 4  , 4
          reflect::align_of<I>(f),                // 4  , 4
          reflect::offset_of<I>(f));              // 0  , 4
      }, f);
    }
    ```
  </Step>

  <Step title="Compile and run">
    Pass the directory containing the `reflect` file as an include path:

    ```bash
    g++ -std=c++20 -I/path/to/reflect-dir hello.cpp -o hello
    ./hello
    ```

    Expected output:

    ```
    foo.a:int=42 (4/4/0)
    foo.b:E=1 (4/4/4)
    ```

    <Tip>
      Add `-DNTEST` to skip the built-in self-tests and cut compile time roughly in half (~0.25s → ~0.11s).
    </Tip>
  </Step>
</Steps>

## What just happened

| Feature | What it does |
| ------- | ------------ |
| `reflect::size(f)` | Returns the number of data members in the aggregate `f`. |
| `reflect::type_name(f)` | Returns `std::string_view` of the type name of `f` or any expression. |
| `reflect::type_id(expr)` | Returns a compile-time value unique to the type of `expr`. Comparable with `==` and `!=`. |
| `reflect::enum_name(e)` | Returns `std::string_view` of the enumerator name for value `e`. |
| `reflect::member_name<N>(f)` | Returns `std::string_view` of the name of the Nth member of `f`. |
| `reflect::get<N>(f)` | Returns a reference to the Nth member of `f`. |
| `reflect::get<"name">(f)` | Returns a reference to the member named `"name"` in `f`. |
| `reflect::to<std::tuple>(f)` | Converts `f` into a `std::tuple` of references to its members. |
| `reflect::for_each(fn, f)` | Calls `fn` with a compile-time index `I` for each member of `f`. |
| `reflect::size_of<N>(f)` | `sizeof` of the Nth member. |
| `reflect::align_of<N>(f)` | `alignof` of the Nth member. |
| `reflect::offset_of<N>(f)` | Byte offset of the Nth member within the struct. |
