`reflect` is a C++20 static reflection library that operates entirely at compile time. All functions live in the `reflect` inline namespace (`reflect::inline v1_2_5`) and are accessed via `#include <reflect>`.

<Note>
  Unless otherwise noted, every function in this library is `constexpr` and `noexcept`. All name lookups are resolved at compile time — there are no virtual dispatch, RTTI, or exception-handling dependencies. The library compiles cleanly with `-fno-exceptions -fno-rtti -Wall -Wextra -Werror -pedantic`.
</Note>

Functions that query properties of a type or value at compile time.

<CardGroup cols={2}>
  <Card title="reflect::size" href="/api/type-introspection#reflectsize">
    Returns the number of data members in an aggregate type.
  </Card>
  <Card title="reflect::type_name" href="/api/type-introspection#reflecttype_name">
    Returns the unqualified string name of a type as `std::string_view`.
  </Card>
  <Card title="reflect::type_id" href="/api/type-introspection#reflecttype_id--type_id_v">
    Returns a compile-time unique integer identifier for a type, derived from its name.
  </Card>
  <Card title="reflect::type_id_v" href="/api/type-introspection#reflecttype_id--type_id_v">
    Inline variable template that provides a unique compile-time identity object per type.
  </Card>
  <Card title="reflect::visit" href="/api/type-introspection#reflectvisit">
    Calls a callable with all members of an aggregate expanded as a variadic argument pack.
  </Card>

## Member Access

Functions that address individual members by index or name.

<CardGroup cols={2}>
  <Card title="reflect::member_name" href="/api/member-access#reflectmember_name">

  </Card>
  <Card title="reflect::get&lt;N&gt;" href="/api/member-access#reflectget-by-index">
    Accesses the Nth member of an aggregate by compile-time index.
  </Card>
  <Card title="reflect::get&lt;&quot;name&quot;&gt;" href="/api/member-access#reflectget-by-name">

  </Card>
  <Card title="reflect::has_member_name" href="/api/member-access#reflecthas_member_name">
    Concept that is satisfied when type `T` has a data member with the given name.
  </Card>
  <Card title="reflect::index_of" href="/api/member-access#reflectindex_of">
    Returns the zero-based index of a named member within an aggregate.
  </Card>
  <Card title="reflect::member_type" href="/api/member-access#reflectmember_type">

## `constexpr` and `[[nodiscard]]` guarantees

Every public function in the `reflect` namespace is marked `[[nodiscard]] constexpr`. The library compiles without exceptions or RTTI, so it works cleanly with `-fno-exceptions -fno-rtti`.

`type_id_v` is an inline variable template (not a function) and has no call overhead at all.

The library verifies its own behaviour via `static_assert` on every include. Pass `-DNTEST` to skip these checks and reduce compile time.
