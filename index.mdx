---
title: "reflect — C++20 Static Reflection"
description: "A single-header, zero-dependency C++20 library for compile-time introspection of aggregate types, enums, and struct members."
---

`reflect` gives you compile-time access to the structure of your C++ types — member names, types, values, and layout — without macros, registration, or runtime overhead. It works with any aggregate type out of the box.

```cpp
#include <reflect>

struct foo { int a; float b; };
constexpr auto f = foo{.a = 42, .b = 3.14f};

static_assert(2 == reflect::size(f));
static_assert("foo" == reflect::type_name(f));
static_assert("a"   == reflect::member_name<0>(f));
static_assert(42    == reflect::get<0>(f));
static_assert(42    == reflect::get<"a">(f));
```

## Why reflect?

<CardGroup cols={2}>
  <Card title="Zero dependencies" icon="box">
    Single header file. Drop it in and `#include <reflect>`. No build system changes required.
  </Card>
  <Card title="Compile-time everything" icon="bolt">
    All introspection happens at compile time via `constexpr` and `static_assert`. Zero runtime cost.
  </Card>
  <Card title="Self-verifying" icon="shield-check">
    Runs its own test suite via `static_assert` on every include. If it compiles, it works.
  </Card>
  <Card title="Compiler-agnostic" icon="microchip">
    Works with gcc-12+, clang-15+, and msvc-19.36+. No `#ifdef` per-compiler workarounds.
  </Card>
</CardGroup>

## At a glance

<Steps>
  <Step title="Include the header">
    Copy `reflect` from the repository into your project and include it:

    ```cpp
    #include <reflect>
    ```
  </Step>
  <Step title="Inspect any aggregate type">
    No registration, no macros. Works on any C++ aggregate:

    ```cpp
    struct Point { double x; double y; double z; };

    // Count fields
    static_assert(3 == reflect::size<Point>());

    // Get field names
    static_assert("x" == reflect::member_name<0, Point>());
    static_assert("y" == reflect::member_name<1, Point>());
    ```
  </Step>
  <Step title="Access members by name or index">
    ```cpp
    constexpr auto p = Point{1.0, 2.0, 3.0};

    // By index
    static_assert(1.0 == reflect::get<0>(p));

    // By name (compile-time string)
    static_assert(2.0 == reflect::get<"y">(p));
    ```
  </Step>
  <Step title="Iterate all members">
    ```cpp
    reflect::for_each([](auto I) {
      std::println("{} = {}", reflect::member_name<I>(p), reflect::get<I>(p));
    }, p);
    // x = 1
    // y = 2
    // z = 3
    ```
  </Step>
</Steps>

<Note>
  `reflect` targets C++20. You need gcc-12+, clang-15+, or msvc-19.36+. See [Installation](/installation) for details.
</Note>
